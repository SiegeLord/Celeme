celeme.capi

This module contains the C api to Celeme. Currently only static linking is supported. It is critical to check for errors after each call to Celeme. Error codes are not used, instead you should check for the error as follows: 
1

const char* error;
CELEME_MODEL* model = celeme_load_model("model.cfg", true);
if(error = celeme_get_error())
    printf("%s\n", error);
